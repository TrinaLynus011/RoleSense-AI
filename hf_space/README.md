# RoleSense AI — Backend API

FastAPI backend for the RoleSense AI candidate ranking platform.

Built for **INDIA RUNS** — Redrob AI × Hack2skill Hackathon.

## Endpoints

- `GET /api/health` — Health check
- `POST /api/signup` — Register
- `POST /api/login` — Login
- `POST /api/rank-candidates` — Rank candidates against a job description
- `GET /api/locations` — Available locations
- `GET /api/analytics` — Candidate pool analytics
